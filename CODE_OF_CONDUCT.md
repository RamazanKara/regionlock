# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org),
version 2.1.

In short: be respectful, assume good faith, and focus on what is best for the
community. Harassment and exclusionary behavior are not tolerated.

Report unacceptable behavior to the maintainer via a private
[security advisory](https://github.com/RamazanKara/regionlock/security/advisories/new)
or direct message. Reports are handled confidentially.

The full text is available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
